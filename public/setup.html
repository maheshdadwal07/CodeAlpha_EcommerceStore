🗄️ Database Setup Options

Option 1: MongoDB Atlas (Cloud Database - Recommended)

  1. Create Account: Go to MongoDB Atlas and create a free account
  2. Create Cluster: Choose the free M0 cluster option
  3. Create User: Set up a database user with username and password
  4. Whitelist IP: Add your IP address (or use 0.0.0.0/0 for testing)
  5. Get Connection String: Copy the connection string from the Connect button
  6. Update .env: Add your connection string to the .env file
MONGODB_URI=mongodb+srv://username:password@cluster0.xxxxx.mongodb.net/ecommerce_store?retryWrites=true&w=majority

Option 2: Local MongoDB Installation

  1. Download: Get MongoDB Community Server from mongodb.com
  2. Install: Follow the installation wizard
  3. Start Service: Start MongoDB service on your system
  4. Default Connection: The app will connect to localhost:27017 automatically
